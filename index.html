Arduino Pong เกมสุดคลาสสิกคุณสร้างเองได้!

ในโปรเจกต์ Arduino Pong นี้เราอยากให้คุณได้รู้ความสนุกสุดคลาสสิกอย่างเกม Pong ที่หลายๆคนรู้จักกันดีแต่ถ้าเป็น Pong ปกติก็จะธรรมดาไป เราเลยได้ทำการเพิ่มตัวละครให้คุณเลือกถึง 3 ตัวละคร 3 ความสามารถที่มีความเฉพาะตัว!!! ใน project นี้ทำให้เราเรียนรู้เรื่องการเขียน code arduino โดยการให้แสดงผลต่างๆผ่านหน้าจอ LED ที่อยู่ใน board และคุณก็สามารถเอา code ที่เราสร้างไป พัฒนาต่อได้เลย!!! และสามารถดูวิดีโอ DEMO ได้ที่ด้านซ้ายนี้แต่ถ้าไม่ขึ้นแนะนำให้กดปุ่ม Link Youtube ข้างบนได้เลย!!!
